# Random Forest MLOps Pipeline

This project demonstrates a full MLOps pipeline using a Random Forest classifier on the Iris dataset.

## Tools Used
- **Git & GitHub**: Version control and repository hosting
- **GitHub Actions**: CI/CD pipeline
- **Docker & DockerHub**: Containerization and image registry
- **Kubernetes (Minikube)**: Container orchestration
- **DVC**: Data version control
- **MLflow**: Experiment tracking
- **Pylint & PyUnit (pytest)**: Code quality and unit testing

## Structure
- `src/`: Training and prediction scripts
- `tests/`: Unit tests
- `k8s/`: Kubernetes manifests
